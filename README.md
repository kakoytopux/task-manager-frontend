# Менеджер задач
---

Реализовано REST API и front небольшого таск менеджера.

---

CRUD для задач:
- GET /api/task
- POST /api/task
- PATCH /api/task/:id
- DELETE /api/task/:id

---

Время написания проекта: 7 часов.

---

Стек на бэке:
- Node.js **(Nest.js)**
- TypeScript
- Mongoose
- MongoDB

Стек на фронте:
- Vue.js
- TypeScript
- Tailwindcss
- Vite
